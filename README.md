# Bag-website
This is a sample bag shop website project.
It is created using HTML, CSS, and Bootstrap.
The website shows different types of bags for users.
It includes pages like Home, Products, About, and Contact.
The design is simple, clean, and responsive.
Bootstrap is used for styling and layout.
Images are added to make the website attractive.
This project is useful for beginners to learn web design.
It demonstrates the basic structure of a shopping website.
